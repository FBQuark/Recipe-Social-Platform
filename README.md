# Recipe-Social-Platform

A social platform for Food aficionados alike built using JavaScript and CSS. The Stack used was MongoDB, React, Postman, and Boostrap. The site has a basic log-in/sign-up system using hashed passwords, then has a user feed showing recent posts. It is important to note that this was developed by a team of qualified people, and I focused on the front-end. 

# Features

* Liking Posts
* Reporting posts for wrongdoing, which can then get looked at by administrators
* Multiple kinds of users, i.e. verified users in which someone can apply and get extra "confidence" in the recipes they would upload

# Notice
This does not have a public site in which you can access it, the entire site was locally hosted throughout development.

# Installation

1. Clone or download this repository.
2. install all required functionality in your IDE (i used VSCode).
3. Create your own mongoDB server and refactor the backend to include your new server settings and keys.
